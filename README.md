## 配置

将你下载的压缩文件解压到桌面，右键***netlogin.conf***文件，使用记事本打开，将***username***，***password***分别改成你的校园网账号，你的校园网密码

![](https://s3.bmp.ovh/imgs/2022/09/13/c54fdc72b348acc1.png)

## **使用**

配置完毕后，在已经连接NCWU网络，但未输入密码的情况下，双击启动NetLogin.exe即可成功链接校园网。

![](https://s3.bmp.ovh/imgs/2022/09/13/894eb2b1fd802aa9.png)

下面提供两种更为便捷的方式，你可以选择使用其中一种。（或两者都选，收获双倍快乐。

注：需WLAN设置自动连接NCWU网络

![](https://s3.bmp.ovh/imgs/2022/09/13/108dcacb2badd86a.png)

### **便捷配置one：快捷方式**

配置完毕后，右键NetLogin.exe，选择**创建快捷方式**，将快捷方式拉至桌面，启动电脑后只需桌面双击一下快捷方式，即刻上网。

### **便捷配置two：定时任务**

配置完毕后，右键此电脑，选择管理，点击任务计划程序，创建任务。在**常规**中，名称取一个你认为有意义的，选择***不管用户是否登录都要运行***，且在***不储存密码选项打钩***，配置选择***你当前使用的系统***。在**触发器**中，点击新建，开始任务一栏***选择发生事件时***，日志选择***系统***，源选择***Netwtw10***，事件ID输入***7021***。在**操作**中，点击新建，操作选择***启动程序***，程序或脚本选择***你下载的NetLogin.exe***，起始于填入***NetLogin.exe所处路径***。在**条件**中，在电源一栏***取消只有在计算机使用交流电源时才启动此任务***，***打钩唤醒计算机运行此任务***。
